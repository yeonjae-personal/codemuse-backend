# sbui

## Project setup
```
npm install
```

Use commands below: `npm run (Command)`

| Command       | Description                                            |
| ------------- | ------------------------------------------------------ |
| dev           | Start devserver.                                       |
| build         | Build for production.                                  |
| preview       | Run the program generated by the production build.     |
| type-check    | Check vue markup.                                      |
| test          | Run vitest                                             |
| coverage      | Output Coverage Report.                                |
